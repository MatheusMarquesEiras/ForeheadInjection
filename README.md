# Instalação local

## Usando Makefile
Para instalar o projeto via Makefile, basta usar o comando:
``` bash
make install
```

## Manualmente
Para instalar manualmente, basta seguir os seguintes passos:
``` bash
python -m venv venv
venv/scripts/activate
pip install -r requirements.txt
```
